# Asistente Personal
Se crea una pagina web que permite ver los siguientes 3 sistemas:
1. Un asistente personal: Control web que permite saber que tareas, pagos y habitos hay pendientes o por analizar. Recordatorio en web para que uno pueda ver los pendientes asi como una pagina de hoy para ver el resumen.
2. El budget que permite llevar un tracking web del presupuesto mes a mes para un control real de gastos, ingresos e inversiones
3. Un portafolio tracker que lleva el rendimiento y distribucion de las inversiones hechas hasta ahora.
